Requisito de Seguridad

Por políticas de privacidad de los navegadores modernos, la función navigator.geolocation solo es accesible a través de conexiones HTTPS seguras.

Geolocalización Mendoza

Para evitar errores de despacho, el buscador está restringido a un Bounding Box (BBOX) que cubre únicamente el Gran Mendoza. Esto filtra resultados irrelevantes de otras provincias.

Mendoza Bounds Configuration

const MZA_BOUNDS = "-69.0436, -33.1092, -68.6416, -32.7483";

Lógica de Precios

El tarificador actual utiliza un simulador de mercado dinámico. En entornos de producción, se recomienda conectar este valor con la API de la Secretaría de Servicios Públicos o un cálculo por km/tiempo. 

// Generación de tarifa estimada (ARS)
userData.price = Math.floor(Math.random() * (3500 - 1800) + 1800);
Dato de Interés

El widget incluye una etiqueta de "Tarifa Garantizada" para aumentar la confianza del usuario durante el Paso 2 de cotización.

Local Storage & Persistencia

Para mejorar el UX, el widget recuerda el nombre del pasajero utilizando el almacenamiento local del navegador.

// Persistencia de identidad
localStorage.setItem('taxigo_name', name);

// Recuperación automática
const savedName = localStorage.getItem('taxigo_name');
